## Source

All emojis should be downloaded from Apple's emoji set for consistent iOS-style appearance.

### How to Get Apple Emoji PNGs

1. Use emoji extraction tools from iOS system fonts
2. Download from emoji repositories (e.g., iamcal/emoji-data on GitHub)
3. Use Emoji PNG services that provide Apple-style emojis
4. Extract from Apple Color Emoji font file

Recommended size: 160x160px or 256x256px PNG with transparency
